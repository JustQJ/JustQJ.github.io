模型的保存

  1. 保存训练好的模型最好load到cpu上再保存。
  2. 使用torch.nn.DataParallel()训练的模型，注意保存时要去掉module模块，不然加载时也需要先用DataParallel()包裹才能加载。

```
不去掉module保存：
torch.save(model.state_dict(), ‘validate.pth.tar’)
加载：
model = torch.nn.DataParallel(model)
state = torch.load(‘validate.pth.tar’)
model.load_state_dict(state)

去掉module并在cpu上保存：
model = model.to(‘cpu’)
torch.save(model.module.state_dict(),’validate.pth.tar’)
加载：
state = torch.load(‘validate.pth.tar’)
model.load_state_dict(state
